เป็นโครงสร้างที่ควบคุมและตัดสินใจว่าจะทำ task ใดในช่วงเวลานั้นๆ และแต่ละช่วงของ event loop ก็มีการจัดการดังนี้


1.การตรวจสอบtaskการ settimout() และ setinterval() ถ้ามีก็จะทำการ execute

2.จัดการใน function callback ต่างๆ

3.รอ task ใดๆ ข้างต้นทำงานเสร็จ

4.ตรวจสอบ I/O หรือ event เมื่อตรวจสอบเสร็จจะทำการเรียก setImmediate callback เพื่อทำงานสคริปต์

5.close event ต่างๆ
